# Ejecutar en desarrollo

## Description
1. Clonar el Repositorio
2. Ejecutar

## Installation

```
$ yarn install
```
3. Tener CLI instalado

```
$ npm i -g @nestjs/cli
```
4. Levantar la base de datos

```
$ docker-compose up -d
```
## Running the app

```
# development
$ yarn run start

# watch mode
$ yarn run start:dev
```
## Stack usado
*MongoDb
*Nest
